One of the hardest things about this project was coding the primitive AI. I wanted the “infected” dots to chase the “non infected” dots. So first I needed a way to compare all the dots to each other at the same time. To do that I used a nested for loop. I am happy that I used a nested for loop for my last project because that experience helped me alot in my chemotaxis project. The second thing I did was I made a if statement to see if there was a non infected dot close enough to an infected dot. If there was then I would find a vector starting this the infected dots and pointing to the non infected dot. This would make the infected dot travel along that vector making it seem that it was chasing that dot.
